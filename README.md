# Take The Graal 🏆
Take The Graal est un jeu développé en collaboration avec Nicolas Bourbasquet, lors du projet final de la spécialité ISN du Baccalauréat Scientifique. Nous avons utilisé le canvas de JavaScript avec les librairies de p5.js, de plus, j'ai également utilisé "[un langage](https://github.com/paraceltus/TakeTheGraal/blob/master/js/deroulement.js)" spécialement conçu pour ce projet.
